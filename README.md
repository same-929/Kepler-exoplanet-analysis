# 开普勒系外行星凌星信号检测

本项目使用开普勒太空望远镜公开的光变曲线数据，通过相位折叠和3-sigma阈值法自动检测系外行星的凌星信号。

## 数据来源
NASA Exoplanet Archive - Kepler-10b (真实行星) ，KOI-1684.01 (假阳性）

## 运行方法
1. pip install -r requirements.txt
2. jupyter notebook kepler_analysis.ipynb
